Apex Header Component

This is a test page demonstrating the self-injecting Apex header component. The header is loaded via a single <script> tag — no other setup required.

Usage

To add the header to any website, just include this script:

<script src="https://your-cdn.com/apex-header.min.js"></script>

That's it! The script automatically:

  • Injects the CSS styles
  • Creates the header HTML
  • Sets up all event handlers
  • Handles mobile menu, dropdowns, and accessibility

Event Handling

Listen for custom events from the header:

document.addEventListener('apex-header:action', (e) => {
  console.log('Action:', e.detail.action);
  // Handle 'contact', 'cart', etc.
});

Features

  • ✅ Pill-shaped desktop navigation
  • ✅ Smooth dropdown menus with fade-slide animation
  • ✅ Mobile drawer with spring easing
  • ✅ Staggered menu item animations
  • ✅ Accordion submenus
  • ✅ Backdrop blur effect
  • ✅ Keyboard accessible (Escape to close)
  • ✅ Focus trap in mobile drawer
  • ✅ Body scroll lock when drawer open
  • ✅ Cart count with animated badge

Customization

Edit the CONFIG object in src/apex-header.js to change menu items, links, and actions. Then rebuild with npm run build.
